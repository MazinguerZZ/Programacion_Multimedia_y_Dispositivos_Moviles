# ACTIVIDAD 2.2 Desarrollo de aplicación calculadora

**Ciclo Formativo de Grado Superior**  
**Desarrollo de Aplicaciones Multiplataforma**  
**Programación Multimedia y Dispositivos Móviles**

---

## 1. Objetivo

Demostrar conocimientos sobre desarrollo de aplicaciones para dispositivos móviles, considerando los siguientes aspectos:

- Análisis y uso de clases que modelan ventanas, menús, alertas y controles para el desarrollo de aplicaciones gráficas sencillas (RA2.b).
- Realizar pruebas de interacción usuario-aplicación para optimizar las aplicaciones desarrolladas a partir de emuladores (RA2.g).

---

## 2. Enunciado

**Individualmente**, realizad la siguiente tarea competencial evaluable para demostrar un conocimiento básico en relación con "Aplicaciones Móviles Android":

1. Cread una aplicación Calculadora para Android, tomando como referencia la de la Ilustración 1 y usando la plantilla `A2.2_TEMPLATE.zip`, que permita:

   - **a)** Mostrar los botones del 0 al 9 de los operandos, los de los operadores de suma, resta, multiplicación y división, el de calcular el resultado, así como el de puesta a cero (que resetee el cálculo).
   - **b)** Mostrar los operandos y operadores de la operación en curso, y el resultado cuando se pulse el botón correspondiente.
   - **c)** Mostrar un objeto tipo switch que permita modificar el estilo (color de fondo) de la pantalla del apartado b.
   - **d)** Mostrar un campo donde proporcionar un posible mensaje de error al usuario.

   > 📱 *Ilustración 1 – Ejemplo de interfaz gráfica para la aplicación de calculadora* *(ver imagen adjunta)*

   - **e)** Utilizar "values" y "drawables" para definir algunos de los componentes.
   - **f)** Llevar a cabo las operaciones básicas de suma, resta, multiplicación y división utilizando la clase `Calculator` proporcionada como parte de la plantilla, conforme el usuario pulse los botones correspondientes a los operandos y operadores.
   - **g)** Mostrar un mensaje de error en caso de que el usuario haga una división por 0.
   - **h)** La aplicación debe adaptarse para que se visualice correctamente tanto en posición vertical como horizontal del emulador.

2. Exportad en un `.zip` el proyecto de Android Studio y subidlo al aula virtual con el siguiente nombre:  
   `DAM2_PMM_UT2_Actividad2_Nombres-alumnos`
