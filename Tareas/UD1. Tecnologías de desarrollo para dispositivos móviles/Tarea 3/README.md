# ACTIVIDAD 1.3 Instalación de IDE para desarrollo en móviles

**Ciclo Formativo de Grado Superior**  
**Desarrollo de Aplicaciones Multiplataforma**  
**Programación Multimedia y Dispositivos Móviles**

---

## 1. Objetivo

Demostrar conocimientos sobre tecnologías de desarrollo para dispositivos móviles, sus características y capacidades, considerando los siguientes aspectos:

- Instalación, configuración y uso de entornos de trabajo para el desarrollo de aplicaciones para dispositivos móviles.
- Identificación de configuraciones que clasifican los dispositivos móviles en base a sus características.
- Descripción de perfiles que establecen la relación entre el dispositivo y la aplicación.

---

## 2. Enunciado

**Individualmente**, realizad la siguiente tarea competencial evaluable, extrayendo capturas de pantalla completa del proceso para su posterior documentación:

1. Proceded con la instalación y configuración del IDE de Android Studio siguiendo los siguientes pasos:
   - Recopilad la información sobre los requisitos de sistema de Android Studio para su instalación en Windows, MacOS y Linux.
   - Usando la máquina virtual con Sistema Operativo Linux proporcionada por el profesor, proceded a revisar que se cumplen los correspondientes requisitos.
   - Instalad en dicha máquina virtual el IDE de Android Studio.
   - Configuradlo, instalando todos los paquetes adicionales requeridos.
   - Verificad que llegáis a la pantalla de bienvenida **"Welcome to Android Studio"**.

2. Elaborad un documento en el que recojáis los pasos del punto anterior, incluyendo capturas de pantalla que hayáis sacado de vuestro proceso de instalación personalizado (no usar capturas que no hayáis extraído vosotros mismos).

3. Subid el fichero al aula virtual con el siguiente nombre:  
   `DAM2_PMM_UT1_Actividad3_Nombres-alumnos`

---

## 3. Evaluación

- La evaluación se realizará conforme a la siguiente lista de cotejo.
- La presentación de trabajos **no genuinos o plagiados** conllevará una calificación de **0 puntos** en la actividad para todos los alumnos involucrados.
- La actividad deberá entregarse antes de la fecha de cierre configurada en el aula virtual para ser evaluada. De no ser así, se puntuará con un 0. Se reabrirá el plazo de entrega para las convocatorias de Ordinaria y Extraordinaria para aquellos alumnos que deban presentarse a ellas.
- Tras la entrega, el docente podrá realizar preguntas a los alumnos para evaluar su grado de contribución a la actividad.
